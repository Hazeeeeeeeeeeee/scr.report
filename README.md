How to install:
Backend
1) Install python3
2) Install python lib: requests, Flask-Cors, Flask

Frontend
3) install nodejs https://nodejs.org/en
4) Open powershell in frontend/my-app
5) run commands:
    npm install
    npm install react react-dom react-router-dom

How to run:
Backend
1) python3 .\app.py

Frontend
1) npm run start
